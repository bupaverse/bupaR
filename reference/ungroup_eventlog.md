# Ungroup event log

Remove groups from event log

## Usage

``` r
ungroup_eventlog(log)

# S3 method for class 'eventlog'
ungroup_eventlog(log)

# S3 method for class 'grouped_log'
ungroup_eventlog(log)
```

## Arguments

- log:

  Eventlog

## Methods (by class)

- `ungroup_eventlog(eventlog)`: Remove groups from event log

- `ungroup_eventlog(grouped_log)`: Remove groups from
  [`log`](https://bupaverse.github.io/bupaR/reference/log.md).
