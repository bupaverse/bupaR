# Life cycle classifier

Get the life_cycle_id of an object of class `eventlog`

## Usage

``` r
lifecycle_id(x)

# S3 method for class 'eventlog'
lifecycle_id(x)

# S3 method for class 'eventlog_mapping'
lifecycle_id(x)
```

## Arguments

- x:

  [`eventlog`](https://bupaverse.github.io/bupaR/reference/eventlog.md):
  Object of class
  [`eventlog`](https://bupaverse.github.io/bupaR/reference/eventlog.md),
  or
  [`mapping`](https://bupaverse.github.io/bupaR/reference/mapping.md).

## Methods (by class)

- `lifecycle_id(eventlog)`: Retrieve lifecycle identifier from eventlog

- `lifecycle_id(eventlog_mapping)`: Retrieve lifecycle identifier from
  eventlog mapping
